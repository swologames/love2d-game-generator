# Assets

Game assets including graphics, audio, and fonts.

## Folder Structure

- **/images** - Sprites, textures, backgrounds
- **/sounds** - Sound effects (.wav or .ogg)
- **/music** - Music tracks (.ogg recommended)
- **/fonts** - Font files (.ttf, .otf)
- **/shaders** - Compiled shader files

## Asset Guidelines

### Images
- Player mecha: 32x32 or 64x64 pixels
- Enemies: 16x16 to 128x128 (bosses)
- Bullets: 4x4 to 16x16 pixels
- Use sprite sheets for animations
- PNG format with transparency

### Sounds
- Keep sound effects short (< 1 second)
- Use .ogg or .wav format
- Sample rate: 44.1kHz recommended
- Create variations for repeated sounds

### Music
- Use .ogg format for better file size
- Loop seamlessly
- Separate intro and loop sections if needed

### Fonts
- Readable at small sizes (14-16px)
- Monospace fonts work well for shmups
- Include license information
