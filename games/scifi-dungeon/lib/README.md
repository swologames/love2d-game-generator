# lib/
Third-party Lua libraries. Each library lives in its own file or subfolder.

## Required Libraries
| Library | Version | Purpose | Source |
|---------|---------|---------|--------|
| `lume.lua` | 2.3.0+ | General Lua utilities (map, filter, merge, etc.) | https://github.com/rxi/lume |
| `anim8.lua` | 2.3.0+ | Sprite animation from quads | https://github.com/kikito/anim8 |
| `flux.lua` | latest | Tween / easing for movement animations | https://github.com/rxi/flux |

Download and place each `.lua` file directly in this folder before first run.
