# assets/shaders
GLSL fragment/vertex shaders for Love2D (`love.graphics.newShader`).

| File | Effect |
|------|--------|
| `scanline.glsl` | CRT scanline + vignette overlay (optional, settings toggle) |
| `bloom.glsl` | Additive glow bloom pass for neon light elements |
| `distortion.glsl` | Wavy UV distortion for Psionic/hazard zones |

See GDD Section 6.5 for shader design notes.
