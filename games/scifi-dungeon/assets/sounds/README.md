# assets/sounds
Short sound effects loaded as `static` Love2D audio sources.

## Naming Conventions
- `step_<surface>.wav` — Footstep variants
- `attack_<type>.wav` — Attack sounds
- `enemy_hurt_<type>.wav` — Enemy damage sounds
- `player_hurt.wav`, `player_death.wav`
- `door_open.wav`, `door_locked.wav`
- `ui_click.wav`, `terminal_boop.wav`
- `item_pickup.wav`, `levelup.wav`
- `heal.wav`, `hazard_damage.wav`

See GDD Section 7.2 for the full sound effects list.
