# assets/fonts
Font files used for the terminal/monospace UI aesthetic.

## Planned Fonts
- `ShareTechMono-Regular.ttf` — Primary UI/HUD font (monospace, Google Fonts)
- `Orbitron-Regular.ttf` — Display/title font (geometric, Google Fonts)

Both are open-source (SIL OFL). Download from https://fonts.google.com before use.
