# assets/images
Sprite sheets, wall textures, portraits, enemy sprites, UI frames, and icon sheets.

## Naming Conventions
- Wall textures: `wall_<type>.png` (64×64 px tileable strips)
- Enemy sprites: `enemy_<name>.png` (base 64×64 px, may include multiple frames)
- Portraits: `portrait_<class>_<variant>.png` (64×64 px)
- UI elements: `ui_<component>.png`
- Icons: `icon_<name>.png` (16×16 px)
