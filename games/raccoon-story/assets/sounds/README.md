# Sounds

Place all sound effects here (.wav or .ogg format).

## Guidelines

- Short, impactful sounds
- Normalize volume levels
- Use descriptive names
- Keep file sizes small

## Categories

- `player/` - Player actions (footsteps, dash, pickup)
- `enemies/` - Enemy sounds (bark, shoo, etc.)
- `ambient/` - Environmental sounds
- `ui/` - Menu clicks, notifications
