# Fonts

Place all font files here (.ttf or .otf format).

## Guidelines

- Use fonts with appropriate licenses
- Rounded, friendly fonts match the cozy aesthetic
- Include regular and bold weights if needed

## Recommended Free Fonts

- Varela Round
- Quicksand
- Fredoka One
- Patrick Hand (handwritten style)

## Usage

Load fonts with different sizes as needed:
```lua
Assets:loadFont("title", "assets/fonts/VarelaRound.ttf", 48)
Assets:loadFont("body", "assets/fonts/VarelaRound.ttf", 16)
```
