# Music

Place all music tracks here (.ogg format recommended for looping).

## Guidelines

- Use OGG Vorbis format for better compression
- Ensure proper loop points
- Keep consistent volume levels
- Name by scene/context

## Tracks Needed

- `menu_theme.ogg` - Main menu
- `home_cozy.ogg` - Den/home base
- `night_adventure.ogg` - Exploration
- `danger_theme.ogg` - Chase sequences
- `success_jingle.ogg` - Victory stinger
