# Images

Place all sprite and image assets here.

## Organization

- `characters/` - Player, enemies, NPCs
- `items/` - Trash items, collectibles
- `environment/` - Tiles, backgrounds, props
- `ui/` - Buttons, icons, HUD elements

## Guidelines

- Use PNG format with transparency
- 32x32 pixels for most entities
- Use consistent naming: `entity_state_frame.png`
- Organize by category in subfolders
